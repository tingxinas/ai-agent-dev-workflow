# Agent 任务计划示例

## 任务目标

构建一个可复用的 Agentic Development Workflow 模板仓库，帮助开发者把 AI Agent 系统性地融入需求分析、任务拆解、代码实现、Debug 和文档交付。

## 任务拆解

### 1. 项目说明

- 编写 README；
- 说明项目背景、目标和价值；
- 展示 Agent 开发流程图；
- 说明适用场景和安全边界。

### 2. 工作流文档

- 梳理需求澄清流程；
- 梳理任务拆解流程；
- 梳理代码生成流程；
- 梳理调试、审核和交付流程。

### 3. 过程记录

- 提供需求模板；
- 提供任务计划模板；
- 提供 Debug 记录模板；
- 提供公开发布检查清单。

### 4. 演示页面

- 创建静态 HTML 页面；
- 展示核心流程；
- 展示工作流价值；
- 展示使用工具清单。

### 5. 最终检查

- 检查文件结构；
- 检查文字是否适合公开；
- 检查是否包含敏感信息；
- 初始化 Git 仓库并准备上传。

## Agent 输出要求

- 语言简洁，适合公开展示；
- 不夸大效果；
- 明确人负责最终审核；
- 不包含账号、密钥、Token 或隐私信息。
