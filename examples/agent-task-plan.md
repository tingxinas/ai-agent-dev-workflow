# Agent 任务计划示例

## 任务目标

构建一个可用于展示 AI Agent 辅助开发能力的 GitHub 项目。

## 任务拆解

### 1. 项目说明

- 编写 README；
- 说明项目背景、目标和价值；
- 展示 Agent 开发流程图；
- 写明效果评估数据。

### 2. 工作流文档

- 梳理需求分析流程；
- 梳理任务拆解流程；
- 梳理代码生成流程；
- 梳理调试和交付流程。

### 3. 证明材料

- 列出截图类型；
- 列出 GitHub 链接说明；
- 列出在线演示地址说明；
- 提供申请表单可复制文本。

### 4. 演示页面

- 创建静态 HTML 页面；
- 展示核心流程；
- 展示效率提升指标；
- 展示使用工具清单。

### 5. 最终检查

- 检查文件结构；
- 检查文字是否适合公开；
- 检查是否包含敏感信息；
- 初始化 Git 仓库并准备上传。

## Agent 输出要求

- 语言简洁，适合公开展示；
- 不夸大效果；
- 数据需要标注为经验估算；
- 不包含账号、密钥、Token 或隐私信息。
