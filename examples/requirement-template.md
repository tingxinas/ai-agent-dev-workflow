# 需求输入模板

## 项目名称

AI Agent 辅助开发工作流

## 背景问题

个人开发或小团队开发中，经常需要同时完成需求分析、代码实现、测试验证和文档整理，流程分散且重复劳动较多。

## 目标用户

- 个人开发者；
- 小团队研发人员；
- 需要快速构建业务工具的人；
- 需要提交 AI 工具使用证明的人。

## 核心目标

- 将模糊需求转化为明确任务；
- 提升初版代码生成效率；
- 缩短 Debug 和问题定位时间；
- 自动整理交付文档和证明材料。

## 功能需求

1. 支持输入自然语言需求；
2. 输出任务拆解清单；
3. 输出推荐技术方案；
4. 辅助生成代码或修改建议；
5. 辅助整理 README 和证明材料；
6. 提供项目效果评估说明。

## 验收标准

- README 能清楚说明项目价值；
- docs 中包含完整工作流；
- examples 中包含可复用模板；
- demo 页面可以展示项目成果；
- 项目适合作为 GitHub 证明材料提交。
