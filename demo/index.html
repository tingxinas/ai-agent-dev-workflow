AI Agent Development Workflow

用 AI Agent 提升软件开发效率

本项目展示一套从需求分析、任务拆解、代码生成、调试修复到文档交付的 AI Agent 辅助开发流程，适合作为 GitHub 展示项目和 AI 工具申请证明材料。

40%-60% 常规功能开发效率提升
约 60% 需求拆解和方案整理时间减少
约 70% README 与交付文档整理时间减少

核心工作流

01
需求输入
将自然语言想法、截图、日志或已有项目说明输入给 Agent。
02
任务拆解
由 Agent 输出功能边界、技术方案、任务清单和验收标准。
03
代码生成
使用 Claude Code、Cursor、Codex 等工具生成初版代码或修改现有项目。
04
调试修复
让 Agent 分析报错日志、定位问题原因，并给出修复路径。
05
文档交付
自动整理 README、使用说明、测试清单和证明材料。
