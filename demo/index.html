Agentic Development Workflow

把 AI Agent 变成稳定的软件开发协作者

一套从需求澄清、任务拆解、代码实现、Debug 分析、人工审核到文档交付的工作流模板。适合个人开发者、小团队、原型项目和内部工具开发。

Prompt Builder

根据项目目标生成可直接复制给 Agent 的结构化提示词。

Agent Fit Score

在执行前评估任务是否适合 Agent 辅助，避免盲目自动化。

Release Guard

发布前检查敏感信息、文档真实性、截图隐私和 License。

核心流程

01
Clarify
澄清目标用户、核心问题、功能边界和验收标准。
输出：需求澄清文档
02
Plan
拆解任务、确定实现顺序、识别风险和测试路径。
输出：任务计划
03
Build
使用 AI 编程工具生成初稿、修改代码和重构重复逻辑。
输出：可运行功能
04
Debug
根据现象、日志和代码定位问题，形成最小验证路径。
输出：修复方案
05
Review
人工检查业务逻辑、安全风险、依赖变化和可维护性。
输出：审核记录
06
Ship
整理 README、运行说明、测试清单和发布检查项。
输出：交付材料
